  • un link/riferimento relativo fa riferimento alla posizione del file HTML della pagina nel file system, ovvero alla cartella dove si trova il file
  • un link/riferimento assoluto è rappresentato da un URL completo, ovvero con la forma https://nomeserver.dominio/path/resource-name
  • Riferimento Legge Stanca 4/2004, che stabilisce 22 criteri che le pagine web devono rispettare per essere considerate accessibili. Per esempio:
    • le immagini che veicolano informazione devono avere un testo alternativo come valore dell'attributo alt
    • il contrasto tra colore di primo piano del testo e colore di sfondo deve essere sufficiente perchè l'utente possa leggere il testo senza difficoltà
    • le animazioni (es. GIF animate) non devono superare una determinata frequenza
    • gli acronimi e le abbreviazioni vanno sciolti (ovvero esplicitati) alla prima occorrenza nella pagina con il tag

Il tag consente di specificare immagini diverse in base alla larghezza del browser.

I documenti/pagine di stile CSS consentono di specificare l'aspetto presentazionale degli elementi di una pagina web. Un file CSS si compone di regole con la seguente sintassi:

selettore/i {
    proprità1:valore1;
    proprità2:valore2;
    ...
}

I selettori possono riferirsi a:

  • tag (si usa semplicemente il nome del tag, per es. h1 {...})
  • singoli elementi identificati dall'attributo id: si usa il # e il valore dell'attributo id dell'elemento che si seleziona, per es.:
<p id='autore'>...<p>

#autore {...}
  • gruppi di elementi identificati dal medesimo attributo class: si usa il . e il valore dell'attributo class degli elementi che si selezionano, per es.:
<p class='importante'>...<p>
...
<h1 class='importante'>...<h1>

.importante {...}

Le regole di stile possono essere scritte:

  • nell'attributo style dei tag
  • nel tag <style> nella pagina HTML
  • in un file CSS separato e linkato alla pagina HTML con:
<link rel="stylesheet" type="text/css" href="css/styles.css" />
