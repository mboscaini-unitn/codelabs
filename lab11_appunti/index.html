Nel 1966 i due informatici italiani Corrado Boehm e Giuseppe Jacopini hanno dimostrato il seguente teorema:

Qualunque algoritmo può essere implementato (realizzato) utilizzando le sole strutture di controllo: sequenza, selezione, iterazione.

Una struttura di controllo controlla il flusso esecutivo di un programma, ovvero stabilisce qual è la prossima istruzione da eseguire.

La selezione banalmente stabilisce la successiva istruzione è quella a destra dell'istruzione appena eseguita, se presente, oppure quella sottostante.

La selezione (o biforcazione, o condizione, o confronto, o scelta) consente di scegliere quale istruzione/i eseguire in base al valore di verità (o booleano) di una condizione.

L'iterazione (o ripetizione, o ciclo, o loop) ripete un numero di istruzioni un determinato numero di volte (for) oppure fin tanto che (while) una condizione è vera.

Come conseguenza del teorema, le tre strutture indicate sono dette strutture fondamentali della programmazione. Inoltre, da tale teorema ha preso avvio la programmazione strutturata. In particolare, è stato sorpassato l'uso del costrutto di salto (goto, jump) che portava alla scrittura di programmi estremamente complicati da seguire nel loro flusso esecutivo (il cosiddetto spaghetti code),

La chiamata di funzione è un altro costrutto che altera il normale (sequenziale) flusso esecutivo.

La programmazione a eventi associa l'esecuzione di codice (normalmente raccolto in una funzione) al verificarsi un evento. Per esempio:

  • onload: quando la pagina è stata caricata nel browser
  • onclick: quando l'utente clicca su un elemento della pagina

HTML5/JavaScript: Numeri random 3

Pagina web per il calcolo di n numeri random tra 1 e n, dove n è inserito dall'utente. Inoltre, definisce la funzione JavaScript isPrimo() per verificare se un numero è primo e elenca quali e quanti sono i numeri primi generati.

Funzioni e metodi JavaScript:

document.getElementById()
document.innerHTML()
Math.floor()
Math.random()
onclick()

Un numero intero è primo se:

  1. è maggiore di 1 (quindi 1 non è un numero primo)
  2. è divisibile soltanto per 1 e per se stesso
/**
 * Ritorna true se il numero è primo, false altrimenti
 * @param {*} num
 * @returns
 */
function isPrimo(num) {
    if (num < 2) return false;

    for (let i = 2; i < num; i++) {
        if (num % i === 0) {
            return false;
        }
    }
    return true;
}
