Il browser ignora i tag non riconosciuti (nell'esercizio il tag <invenzione>), quindi che non sono tag HTML validi.

<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Unitn - Page Title</title>
    </head>
    <body>
        <h1>This is a Heading</h1>
        <hr>
        <p title="I'm a tooltip">This is a paragraph.</p>
        <invenzione lang="it">Testo inventato</invenzione>
    </body>
</html>
  • il tag <br> (break rule) inserisce (marca) un'interruzione nel contenuto, che normalmente viene "resa" con un "a capo".
  • il tag <hr> oppure <hr /> (hard rule) inserisce (marca) un'interruzione nel contenuto, che normalmente viene "resa" con una riga, o linea orizzontale.

Tre livelli di un documento:

  • testo = puro testo (plain text)
  • ipertesto = testo + collegamenti (link), ottenuti grazie al tag <a>
  • ipermedia = testo + collegamenti + contenuti multimediali ottenuti tramite appositi tag, per esempio per inglobare (embed) un'immagine utilizzo il tag <img>

È simile al principio base del problem solving della scomposizione di un problema in sottoproblemi più semplici da risolvere.

Si tratta di un'applicazione del motto divide et impera.

I font si distinguono in:

  • font sans-serif (senza grazie), detti anche "a bastoni"
  • font serif

Tra i sistemi di misura per la grandezza dei font, troviamo:

  • px = pixel (valori assoluti)
  • % (valori relativi)
  • em (valori relativi alla dimensione base del carattere definita nel browser oppure nel documento)

Crea una pagina web (1984a/index.html) con l'incipit del libro 1984 di George Orwell. Marca opportunamente alcune parti del contenuto, applicando i seguenti tag all'interno del tag <body>:

  • <h1>, <h2> (, <h3>)
  • <p>
  • <br>
  • <hr>
  • i tag di formattazione: <b>, <i>, <strong>, <em>, <mark>, <ins>, <del>, <sup>, <sub>, <small>

Inoltre:

  • indicare la fonte con il tag <a>
  • includere con il tag <img> un'immagine di una locandina di un film tratto dal libro oppure una copertina di un'edizione del libro
